# 🐸 Doodle Jump Clone (Python)

A fast-paced vertical platformer built with Pygame.  
Jump, survive, and reach the highest score!

![Gameplay](images/gameplay.gif)

## ✨ Features
- Endless jumping gameplay
- Random platform generation
- Score system

## 🎮 Controls
- A / ← → Move left
- D / → → Move right

## ⚙️ Installation
pip install pygame  
python main.py
